# Frontend module

This folder groups the Angular client application.

## Contents

- `ecom-ui/` - Angular 17 application
- `.gitignore` - frontend-level ignore rules

## Purpose

The frontend consumes the backend API and provides:
- public product browsing
- login
- cart and order flows
- admin catalog management UI
- admin analytics dashboard UI
